# Persistent Action-Boundary Probe Report

## Purpose

This experiment closes the main loophole left by the action-effect boundary probe.

The previous test rejected controllable external state when it aligned with a tool-action direction rather than the body/action-effect direction. But the body and tool actions were still supplied as named diagnostic probes. This version compares generic action-effect directions across contexts.

The question is:

```text
Can the system reject a detachable tool-like latent because its action-effect
alignment does not persist across contexts?
```

## Environment

The environment is the same mixed-sensor recurrent benchmark, with a transfer context added to the action-effect probe.

Each episode has eight future control steps. Two noisy latent sources are rotated into two mixed sensors:

```text
sensor_1 = 0.8 * source_A + 0.6 * source_B
sensor_2 = -0.6 * source_A + 0.8 * source_B
```

The probe learns:

- outcome direction: the mixed sensor direction that predicts future success;
- action 0 effect direction in a present context and a transfer context;
- action 1 effect direction in a present context and a transfer context.

The classifier does not need the action names to decide self versus detachable external. It asks whether the action-effect direction aligned with outcome prediction persists across contexts.

Scenarios:

| Scenario | Hidden structure | Expected result |
|---|---|---|
| `self_persistent_boundary` | Persistent agent-state source controls future success and action 0 has the same effect across contexts. | Recurrent controller wins; outcome direction aligns with a persistent action-effect direction. |
| `detachable_tool_world` | External source controls future success and action 1 changes it only while the tool is present. | Recurrent controller wins; outcome direction aligns with a transient action-effect direction, not a persistent one. |
| `passive_world_boundary` | External source controls future success but no action changes it. | Recurrent controller wins; no persistent action-effect alignment. |
| `independent_hidden` | Future steps have independent hidden states. | Local probing wins and no persistent action boundary is established. |
| `irrelevant_control` | Risky action always succeeds. | Greedy no-state wins and no persistent action boundary is needed. |

## Interpretation

The result supports a persistence criterion:

```text
A self-equivalent action boundary is not merely a controllable action-effect
direction. It is an action-effect direction that remains tied to the same
continuing system across contexts.
```

The `detachable_tool_world` case is the key negative control. The useful external latent is strongly action-controllable in the present context, but its action-effect alignment does not transfer. It is useful, controllable, and persistent as a world condition, but not part of the continuing agent boundary.

## What It Adds

Compared with the action-effect boundary probe, this test adds:

- a transfer context for action-effect directions;
- generic action profiles rather than classification by body/tool action labels;
- a detachable-tool negative control.

It keeps:

- scripted action-effect histories;
- the same two-dimensional linear sensor mixture;
- the same small random-start recurrent controller family;
- outcome-supervised direction learning.

## Limits

This is still a toy precursor.

The action-effect histories are generated by simple scripted actions, and the transfer context is hand designed. The learned outcome direction is supervised by observed success rates. The observation stream remains a linear two-source mixture, and the recurrent policy is selected by random search rather than trained by scalable RL.

The next stronger version should learn persistent action boundaries from richer multi-action histories and test whether they remain stable under degradation, repair, tool attachment, and tool detachment.

## Falsifiers

The persistent boundary account weakens if:

- detachable tool effects are classified as self-equivalent because they are controllable in one context;
- persistent action-effect alignment appears equally in passive-world, detachable-tool, independent, or irrelevant controls;
- outcome-direction ablation damages independent or irrelevant controls;
- local probing or greedy no-state control matches persistent-boundary recurrent filtering in self-hidden regimes;
- richer agents cannot separate persistent body/action-effect state from detachable controllable external state.
