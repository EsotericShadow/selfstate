# SSRM-3D Recurrent Observer Report

## Question

Can learned recurrent observers recover self-equivalent state from embodied 3D action-observation traces, without making the LLM the controller?

The first SSRM-3D experiment used a hand-built self-state workspace. This follow-up trains PyTorch observers on traces from the same world and asks whether their hidden states recover agent-bounded variables:

- energy;
- integrity;
- mobility;
- sensor capability.

The observer is not asked whether it has a self. It is tested by hidden-state probes, future-viability prediction, self-subspace ablation, and counterfactual self-state edits.

## Why This Matters

The hidden-state loophole says:

> Any system with persistent hidden variables benefits from modeling those variables.

This experiment narrows the loophole. A hidden state counts as self-equivalent only if it is:

- learned from action-observation traces;
- decodable as agent state;
- stronger than a stateless frame model when pressure accumulates;
- connected to future viability;
- editable in the direction expected for energy, integrity, and capability.

## Command

```bash
python3 experiments/ssrm_3d_recurrent_observer.py --episodes-per-stage 42 --ticks 540 --seed 20260608 --hidden-size 32 --epochs 180 --batch-size 64 --learning-rate 0.004 --device auto
```

The canonical run used Apple Silicon MPS:

```text
device mps torch 2.8.0 mps_available True
```

## Learners

The experiment trains four observers on the same trace set:

| Architecture | Role |
|---|---|
| `frame_mlp` | Stateless frame-only control. |
| `torch_rnn` | Simple recurrent observer. |
| `torch_gru` | Gated recurrent observer. |
| `torch_lstm` | Gated recurrent observer with cell state. |

Training traces are generated by all SSRM-3D source agents:

- `reactive_no_self`;
- `world_model_only`;
- `layered_self_state`;
- `layered_self_state_ablation`.

This prevents the learned observer from only imitating the hand-built self-state agent.

## Tests

For each stage, the experiment measures:

- `self_probe_r2`: linear decodability of energy, integrity, mobility, and sensor capability from hidden state;
- `world_probe_r2`: decodability of local world variables;
- `recurrent_self_gain_over_frame`: recurrent self-state gain over the stateless frame model;
- `self_ablation_penalty`: future-viability error increase after removing the learned self subspace;
- `world_ablation_penalty`: matched removal of the learned world subspace;
- `self_edit_future_swing`: future-viability change after counterfactually moving hidden state along the learned self-state direction.

## Results

| Stage | Pressure | Best recurrent | Frame self R2 | Recurrent self R2 | Gain | Self ablation penalty | Self edit swing | Verdict |
|---:|---|---|---:|---:|---:|---:|---:|---|
| 0 | Spatial resources | `torch_lstm` | 0.747 | 0.780 | 0.032 | 0.039 | 0.008 | Body state is decodable, but recurrence adds little in the low-pressure task. |
| 1 | Hidden energy | `torch_gru` | 0.803 | 0.873 | 0.071 | 0.229 | 0.330 | Hidden energy recruits editable self-state. |
| 2 | Body drift | `torch_gru` | 0.615 | 0.814 | 0.199 | 0.210 | 0.330 | Recurrent self-state becomes useful under capability drift. |
| 3 | Delayed options | `torch_gru` | 0.481 | 0.843 | 0.362 | 0.177 | 0.330 | Future-option pressure strengthens recurrent self-state. |
| 4 | Commitment recovery | `torch_gru` | 0.594 | 0.837 | 0.244 | 0.155 | 0.330 | Commitment pressure preserves the learned self-state role. |
| 5 | Subsystem arbitration | `torch_gru` | 0.535 | 0.838 | 0.303 | 0.144 | 0.330 | Arbitration pressure preserves the learned self-state role. |
| 6 | Multiagent social | `torch_gru` | 0.638 | 0.862 | 0.224 | 0.155 | 0.330 | Social pressure preserves the learned self-state role. |

All seven stage verdicts pass the current precursor criteria.

## Interpretation

The result is not that a recurrent hidden state is automatically a self.

Stage 0 is the control: body state is decodable, but the recurrent gain is only 0.032 and the counterfactual self-edit swing is only 0.008. That supports the pressure-gradient interpretation rather than a blanket hidden-state interpretation.

Stages 1-6 show the stronger pattern: as hidden energy, body drift, delayed options, commitments, arbitration, and social pressure accumulate, recurrent observers recover stronger agent-state variables than the stateless frame model. Removing the learned self subspace damages future-viability prediction, and moving the hidden state along the learned self direction changes future-viability prediction.

This is still a precursor. It shows learned observer state, not autonomous learned control.

## Limits

- The trace generator is still the SSRM-3D toy world.
- The source policies are supplied rather than learned end to end in this experiment.
- The future target is future viability, so self-subspace ablation is partly aligned with the target by design.
- Counterfactual edits use learned linear probe directions, not discovered semantic variables.
- The result does not imply consciousness.

## Stronger Next Test

The next step is to train recurrent or model-based controllers in the SSRM-3D world and probe their policy state directly. The stronger test should ask whether the same self-equivalent latent appears when the agent must learn both representation and control from return.
