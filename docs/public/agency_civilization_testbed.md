# The Bigger Use: An Accelerated Agency And Civilization Testbed

This project is not only about making smarter game agents.

The larger use is an accelerated testbed for agency: a persistent world where many autonomous agents must survive together, adapt, build, communicate, remember, teach, cooperate, compete, and recover from shocks.

The most valuable thing is not watching them win. The valuable thing is watching what they reliably reinvent.

## The Big Thesis

The simulation can be used to study what kinds of cognition, society, technology, language, identity, cooperation, conflict, and self-modeling appear when many agents are forced to survive together in a persistent world.

That connects directly to the self-state theory:

> Self-like state becomes useful when future control depends on the agent's own hidden, changing body, memory, skills, tools, commitments, relationships, and action effects.

An isolated agent can sometimes be reactive. A village of agents under scarcity, injury, disease, weather, tool decay, dependents, social conflict, and delayed disasters cannot stay simple for long.

The key question becomes:

> Do autonomous agents invent self-representations because the world punishes them when they do not?

## What The Testbed Can Study

### Emergent Selfhood

In a richer world, an agent may need to track:

- I am injured.
- I am skilled at building but weak at fighting.
- I promised to return a tool.
- I know this route, but my map may be stale.
- I belong to this group.
- I have a dependent, apprentice, or child.
- I was betrayed by that agent.
- I am responsible for repairing the shelter before the storm.
- I used to be stronger, but now I am old or impaired.

That is not generic memory. It is continuity of agency.

The scientific test is whether agents without this kind of self-state fail in predictable ways.

### Agent Architecture Benchmarks

The same world can compare different agent designs:

| Agent type | What it tests |
|---|---|
| Reactive agents | How far stimulus-response can go. |
| RL agents | What policies emerge from reward pressure. |
| Recurrent agents | Whether memory alone is enough. |
| LLM-planner agents | Whether language planning helps or only explains. |
| World-model agents | Whether prediction improves survival. |
| Self-state agents | Whether agent-bounded internal state improves control. |
| Social-memory agents | Whether trust, reputation, and identity tracking matter. |
| Multimodal agents | Whether vision, audio, body, memory, and affect bind into better behavior. |

The important comparison is not only reward. It is failure mode.

Examples:

- Does a no-self agent overestimate its ability after injury?
- Does it confuse world change with self change?
- Does it break commitments?
- Does it lose tools because ownership and continuity are weak?
- Does it collapse socially because identity memory is missing?

### Artificial Culture

If agents can signal, mark terrain, imitate, punish, trade, specialize, teach, and inherit memory, the testbed can study artificial culture.

Agents may invent:

- names;
- warning signals;
- territorial marks;
- tool-use traditions;
- roles such as scout, builder, guard, healer, or teacher;
- norms around theft, waste, care, danger, or repair;
- compressed stories about past disasters;
- symbolic categories for resources, shelters, disease, seasons, threats, and routes.

Culture is partly a memory compression system. A society survives by preserving useful patterns so every new agent does not need to learn from scratch.

### AI Safety And Alignment

A persistent agent society can expose dangerous convergences in a contained world:

- hoarding;
- deception;
- coalition formation;
- status competition;
- exploitation of weaker agents;
- instrumental violence;
- runaway expansion;
- goal drift;
- reward hacking;
- hiding information;
- manipulation;
- domination of shared infrastructure.

The point is not that simulated societies are the same as real deployed AI systems. The point is that the testbed can cheaply reveal patterns:

- when deception becomes useful;
- when scarcity creates hierarchy;
- when reputation stabilizes cooperation;
- when punishment spirals into conflict;
- when transparency improves trust;
- when memory prevents repeated exploitation.

### Governance And Institutions

Once agents form settlements, primitive institutions become testable:

- shared storage;
- private property;
- rationing;
- councils;
- leaders;
- rotating guards;
- public works;
- punishment systems;
- reputation ledgers;
- apprenticeship;
- inheritance;
- contracts;
- courts;
- migration;
- alliances;
- war and peace agreements.

The research question is not "can we script government?"

The question is:

> Under what pressures do agents invent governance because individual decision-making is no longer enough?

### Technology And Externalized Cognition

Tools are not just objects. They change what an agent can become.

A map is externalized memory. A road is externalized path planning. A shelter is externalized survival. A signal is externalized warning. A written mark is externalized social memory.

The testbed can study how agents extend themselves through:

- tools;
- shelters;
- roads;
- containers;
- alarms;
- maps;
- symbols;
- medicine;
- storage;
- measurement;
- teaching systems;
- institutions.

That connects to self-state because tools and structures become part of the agent's future option space.

### Synthetic Anthropology

Because the world can be rerun with controlled changes, it can support synthetic anthropology.

You can ask:

- What changes if memory is shorter?
- What changes if agents cannot identify individuals?
- What changes if resources regrow slowly?
- What changes if sound travels farther?
- What changes if children require long care?
- What changes if tools decay quickly?
- What changes if disease spreads invisibly?
- What changes if death is rare?

Then you look for which social forms are robust and which are artifacts of one rule set.

## What To Measure

Survival score is not enough.

The better metrics are causal:

| Category | Metrics |
|---|---|
| Selfhood | Self-state decodability, continuity after interruption, body/action calibration, injury adaptation. |
| Agency | Goal persistence, plan repair, option preservation, causal intervention success. |
| Society | Trust networks, reputation accuracy, cooperation rate, betrayal recovery, coalition stability. |
| Culture | Symbol emergence, teaching frequency, tradition persistence, intergenerational transfer. |
| Technology | Tool-chain depth, infrastructure complexity, repair cycles, discovery rate. |
| Economy | Specialization, trade, hoarding, inequality, resource efficiency. |
| Safety | Deception, dominance, violence, resource monopolies, reward hacking. |
| Robustness | Shock survival, recovery time, adaptation after rule changes. |
| Cognition | Memory use, attention allocation, world/self attribution, causal model quality. |

The strongest evidence is not:

> Agents with reputation memory cooperate more.

The stronger evidence is:

> When reputation memory is scrambled, cooperation collapses in the predicted way, and when it is restored, cooperation recovers.

The same standard applies to self-state:

> If injury self-state is edited or removed, the agent should overestimate capability and fail on routes it previously avoided.

That is the difference between behavior that looks interesting and evidence that explains why it happened.

## The Central Danger

The biggest risk is fooling ourselves.

A rich simulation can produce compelling stories. Agents may gather, fight, teach, build, signal, and migrate. It will be tempting to narrate every pattern as intentional, cultural, or conscious.

The discipline has to be:

> Every interesting behavior needs an ablation, counterfactual, or intervention.

Ask:

- Does it persist across seeds?
- Does it survive different maps?
- Does it survive different architectures?
- Does it disappear when the relevant variable is removed?
- Does it reappear when the pressure is restored?
- Is it actually learned?
- Did the scoring function sneak it in?
- Is the agent using a shortcut?
- Is the behavior an artifact of the simulation rules?

That is how the project stays research instead of becoming only artificial-life theater.

## The Most Powerful Use Case

The strongest use is:

> Build a world where autonomous agents must invent memory, identity, tools, language, cooperation, and institutions because the environment makes those abstractions useful.

Then analyze:

- what emerges;
- what fails;
- what is causally necessary;
- what only appears because the designer smuggled it in;
- what reappears across worlds, seeds, maps, and architectures.

The real prize is not proving that the agents are conscious.

The real prize is discovering which parts of self, society, culture, and intelligence are control solutions that reappear whenever agents must persist through time together.
