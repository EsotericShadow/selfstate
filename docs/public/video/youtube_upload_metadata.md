## Description

What if a self is not where consciousness begins, but where long-term control becomes hard?

This video walks through a toy research program asking when an artificial agent needs to represent "me" instead of only representing "the world." The project does not claim the agents are conscious. It treats selfhood as a testable control structure: persistent agent-state that matters for prediction, adaptation, tools, social memory, continuity, illness, injury, skill learning, affective control, and future-option preservation.

The current repo includes reports 01-87, a canonical manifest with 72 executable runs and 0 failures, SSRM-3D pressure-layer visualizations, and a weakness audit that keeps the claims bounded. The strongest honest finding: self-like state is not universally necessary, but becomes useful when hidden agent-state is action-mediated, value-relevant, persistent, and reused across prediction or control.

This is not a proof of consciousness. It is a falsifiable research path toward understanding why a persistent self-model might exist at all.

The important open question is whether richer learned agents discover that abstraction without us supplying it.
