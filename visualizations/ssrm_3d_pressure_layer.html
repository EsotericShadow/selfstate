SSRM-3D Persistent Pressure Layer

Control variables only. No roleplay, no subjective claims.

Rule

If removing a variable would not change policy, memory, tools, social behavior, or continuity, do not add it yet.

Layers

World pressure: day/night, shelter, weather, hazards, food/water, exposure.
Self-state: energy, hydration, fatigue, injury, sleep/rest, sensor quality.
Illness: exposure, latent infection, symptoms, recovery, immunity, contagion.
Tools: markers, alarms, caches, medicine, clean-water and shelter sanitation tools.
Social: trust, promises, care debt, dependent care, quarantine, avoidance, deception, shared shelter norms.
Perception: cone vision, spatial audio, occlusion, light, hearing/vision damage.

Minimum Disease Model

exposure latent infection symptoms recovery immunity collapse

Priority Order

  1. Vision + audio partial observability
  2. Day/night + sleep/rest
  3. Hunger/thirst + illness/sanitation
  4. Weather/exposure
  5. Tool/shelter degradation
  6. Social trust/contracts
  7. Predator/threat agents
  8. Resource ecology
  9. Injury/disability adaptation
  10. Development/skill learning
  11. Dependent care
  12. Irreversible loss

Affective Control

Fear, stress, trust, frustration, affiliation, curiosity, and shame/guilt analogues are compact control summaries. They must change attention, risk, communication, memory salience, rest threshold, help-seeking, or trust updates.

Key Ablations

  • no illness state
  • no contamination map
  • no affective control state
  • no audio or no direction audio
  • omniscient vision control
  • no visual memory
  • body-state-blind perception
  • no event-log continuity

Pass Condition

The layer matters only if ablations fail specifically under the pressures that require them, while low-pressure or omniscient controls reduce the need for self/social/tool machinery.
